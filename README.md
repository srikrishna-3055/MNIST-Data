Multi-Model Classification – ML Mini Project

This project builts the multiple machine learning classifiers on the MNIST dataset and It's also identifies the best model using ROC–AUC analysis and further improves it using GridSearchCV hyperparameter tuning.

Project Features: Load & preprocess MNIST dataset ,Train 9 ML models:

KNN

Naive Bayes

Logistic Regression

Decision Tree

Random Forest

AdaBoost

Gradient Boosting

XGBoost

SVM (RBF Kernel)

✔ Compute:

Accuracy

Confusion Matrix

Classification Report

ROC-AUC score for each model

Identify the Best Models Using AUC&ROC.Tune Best Model (SVM) using GridSearchCV. Retrain SVM using best parameters and Finalize ROC curve plot and final accuracy in ML Mini Project.
